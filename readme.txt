Add Testimonials section in your website using [testimonial] shortcode.

== Description ==

This plugin allows you to add Testimonials section using [testimonial] shortcode in posts, pages or widgets (i.e.: sidebar, footer) of your WordPress website.

== Features ==

* Thumbnail as Featured Image
* Name as Title
* Paragraph as Testimonial

== Installation ==

Install the WP7 Testimonials plugin via WordPress Dashboard (from plugin directory or from your computer), or upload the files manually to your server and activate the plugin.

= Install via WordPress dashboard =

*From Plugin Directory*

* Visit ‘Plugins > Add New’
* Search for ‘WP7 Testimonials’
* Install and Activate the plugin
* Use shortcode ‘[testimonial]’ to add the Testimonial Section in a Page.

*From Your Computer*

* Download the WP7 Testimonials plugin
* Visit ‘Plugins > Add New > Upload Plugin’
* Choose the plugin file in .zip format
* Click the Install Now button, and Activate after installation
* Use shortcode ‘[testimonial]’ to add the Testimonial Section in a Page.

= Install via Uploading Files Manually =

* Download the WP7 Testimonials plugin and unzip the folder
* Upload the folder to your ‘/wp-content/plugins/’ directory
* Activate the WP7 Testimonials Plugin from Dashboard
* Use shortcode ‘[testimonial]’ to add the Testimonial Section in a Page.

== Frequently Asked Questions ==
